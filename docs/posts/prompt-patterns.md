# 提示词模式：常用模板

## 1. 任务拆解模板

```text
你是一个资深工程助手。
请把下面任务拆解为最小可执行步骤，并按优先级排序。
输出格式：
1) 目标
2) 步骤清单
3) 风险点
4) 验收标准
任务：{{task}}
```

## 2. 代码审查模板

```text
请只关注以下三类问题：
1) 可能导致错误结果的 bug
2) 安全风险
3) 明显性能问题
输出要求：
- 按严重程度排序
- 每条包含：位置、问题、修复建议
```

## 3. 复盘总结模板

```text
请根据以下实验记录，输出简洁复盘：
- 本次目标
- 做对了什么
- 做错了什么
- 下次要改什么
实验记录：{{notes}}
```

## 使用建议

- 先固定输出格式，再迭代内容细节
- 每个模板只解决一个问题，不要过度混合
- 把高频模板沉淀到自己的片段库
