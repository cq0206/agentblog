# 第一篇：从 0 到 1 搭建你的学习型 Agent

这篇文章记录一个最小可运行的学习型 Agent 搭建过程。

## 1. 目标定义

先把目标写清楚，避免“做了很多但没有进展感”：

- 输入：一个问题或任务描述
- 输出：可执行答案、步骤或代码
- 评估：回答是否正确、是否可复现、是否节省时间

## 2. 最小能力集

建议先实现这 3 个能力：

1. 对话理解：把用户问题转成可执行任务
2. 工具调用：读取文件、执行命令、检索信息
3. 结果复盘：总结输出质量与改进点

## 3. 每周学习节奏

- 周一到周三：学习一个新概念并做最小实验
- 周四到周五：把实验沉淀成模板或脚本
- 周末：写复盘文章，更新路线图

## 本地启动

```bash
npm install

```
